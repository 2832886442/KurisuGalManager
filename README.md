# KurisuGal — Galgame 管理器

基于 **Tauri 2 + React + Vue3 + Rust** 构建的桌面端 Galgame（视觉小说）管理工具。

![QQ20260721-151847](.\Screenshoot\QQ20260721-151847.png)

## 功能

- **游戏库管理** — 添加/编辑/删除游戏，支持名称、别名、路径、分类、标签、简介、封面
- **网格/列表双视图** — 切换布局，分页浏览（12/24/48/96 每页）
- **搜索过滤** — 按名称/别名搜索，侧边栏分类/标签筛选
- **一键启动** — 启动游戏并实时追踪运行状态，自动记录游玩时长
- **游玩状态** — 未游玩 / 游玩中 / 已通关 / 搁置 四种状态，卡片悬浮快捷切换
- **Bangumi 数据集成** — 搜索 Bangumi 自动填充名称、封面、简介、标签
- **拖拽排名** — 自定义等级颜色与名称，拖拽游戏卡片进行排名分类
- **收藏系统** — 收藏夹与快速筛选
- **数据概览** — 可视化统计图表：游玩时长、分类分布、最近游玩
- **游戏截图** — 添加/删除截图，点击预览大图
- **数据管理** — 备份/恢复 JSON 完整数据，可配置存储路径并支持迁移
- **三主题** — 亮色 / 暗色赛博朋克 / 跟随系统
- **界面个性化** — 窗口圆角、缩放比例、每页数量、默认视图、关闭行为

## 技术栈

| 层级   | 技术                             |
| ---- | ------------------------------ |
| 桌面框架 | Tauri 2.11                     |
| 后端   | Rust 2021 Edition（Tokio 异步运行时） |
| 前端   | React 18 + Vue 3 混合架构          |
| 构建   | Vite 5                         |
| 数据存储 | 本地 JSON + 独立封面文件               |

## 快速开始

```bash
# 安装依赖
npm install

# 开发模式
npm run tauri dev

# 生产构建
npm run tauri build
```
